Auto-Invite Admin — Privacy Policy

Effective date: 2025-08-13

Dados acessados

Dados do evento atual no Google Calendar (ID, convidados/attendees, organizador). Não copiamos dados para servidores próprios; a execução ocorre no Apps Script do domínio.

Uso dos dados

Exclusivamente para verificar se cassio@cassioleite.dev já é convidado e, se não for, adicioná-lo. Gravamos a propriedade privada extendedProperties.private.autoInvitedAdmin = "1" para evitar reprocessamento.

Retenção

Sem banco externo. Logs de execução seguem a retenção padrão do Google.

Compartilhamento

Não compartilhamos dados com terceiros.

Segurança

O acesso é mediado pelos escopos do Google. Opera somente dentro do domínio cassioleite.dev.

Contato

E-mail: cassio@cassioleite.dev
